Auth
When a user logs in, we need to answer two questions:
Are the credentials correct?
If yes, how do we prove in future requests that this user is authenticated?
In a database token system:
the password is verified once at login
then the server generates a random token
stores that token in the database with user_id
the client sends it in the Authorization header

the server checks whether that token exists and is still valid
create a secret_key => python -c "import secrets; print(secrets.token_hex(32))"
